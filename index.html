20-Command Suite

Flagship tools.
Four that change everything.

Part of a comprehensive 20-command Civil 3D suite. These four core tools tackle the highest-impact bottlenecks in any land development workflow.

PIPEMAGIC

Pipe Magic

Automatically detects crossing pipe networks and projects them into profile views. No more manually hunting for pipe conflicts — PIPEMAGIC handles the spatial analysis and annotation instantly.

  • Automatic detection of all pipe network crossings
  • Projects crossing pipes directly into profile views
  • Works across multiple network types simultaneously
BULKSUR

Bulk Surface Profile Creator

Batch-creates surface profiles and profile views for multiple alignments simultaneously. Eliminate the repetitive one-by-one workflow and process entire project corridors in seconds.

  • Select all alignments at once — profiles created in batch
  • Consistent profile view formatting across the project
  • Saves hours on multi-alignment corridor projects
ALIGNDEPLOY

Align Deploy

Deploys copies of a cross alignment along a main alignment at defined intervals. Perfect for road corridors, utility easements, and subdivision layouts that require repeated cross-section geometry.

  • Set interval spacing and deploy all copies at once
  • Works on curved and straight main alignments
  • Ideal for road cross-sections and utility corridor layouts
INVERTPULLUP

Invert Pull Up

Calculates invert elevation at any point along a pipe and labels it directly in the drawing. Eliminates manual math and drafting errors when documenting pipe inverts for construction documents.

  • Pick any station point — invert calculated instantly
  • Label placed automatically with correct elevation
  • Eliminates manual station math and drafting errors

Getting Started

Up and running in minutes

Install once — all four tools load automatically into Civil 3D's ribbon.

01

Download & Install

Run the Inno Setup installer. It deploys the bundle to your Civil 3D 2026 application plugins folder automatically.

02

Open Civil 3D

Launch AutoCAD Civil 3D 2026. The Advanced Land Dev Tools ribbon tab appears automatically — no manual loading needed.

03

Run Commands

Click any tool button in the ribbon — or type the command (BULKSUR, ALIGNDEPLOY, PIPEMAGIC, INVERTPULLUP) directly in the command line.

About the Founder

Built by a civil engineer,
for civil engineers.

I'm Juan Felipe Cadavid Herrera, a civil engineer specializing in land development. After years of watching project timelines expand because of repetitive, manual Civil 3D workflows, I built ALDT to solve the problem from the inside.

Every tool in this plugin was designed to address a real bottleneck I encountered in the field — from batch-creating hundreds of surface profiles to instantly labeling pipe inverts that would otherwise require manual station math.

ALDT is the result of combining civil engineering expertise with modern .NET development — purpose-built for AutoCAD Civil 3D 2026 and designed to fit seamlessly into your existing workflow.
